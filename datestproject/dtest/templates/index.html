Intro

Welcome everyone to EDATEST.

EDATEST your data speaks.

EDATEST means Exploratory Data Analysis Test.

Through this site, you can easily explore and analyze your data.

This site provides a great service for everyone, both analysts and non-analysts.

Try for yourself and test it out.

Work

EDATEST just choose your data and click a button to get the result.

1- Number of rows, columns and missing data.

2- Info: The information contains the number of columns, column labels, column data types, memory usage, range index, and the number of cells in each column (non-null values).

3- Describe: Description of the data, If the Data contains numerical data, the description contains these information for each column: count, mean, std, min, 25%, 50%, 75%, max.

4- Head: The first 5 rows from top data.

5- Tail: The last 5 rows from bottom data.

6- Correlation: The measure of the linear relationship between the two variables in data.

7- Heatmap: A graphical representation of data where each value of a matrix is represented as a color.

About

Simply put, at EDATEST we believe that every Data Administrator can understand and analyze their own data.

Our goal is to remove any technical barriers that can prevent you from exploring your data. Our powerful tools enable everyone to analyze and test data. Whether you're a beginner or analysis expert, we're excited to help you on your journey!

After seeing an increased need for data, the EDATEST builder was born. We wanted to offer a data exploration platform that would require no coding skills or analysis experience.
